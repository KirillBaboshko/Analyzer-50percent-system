ШАГ 1: Создайте репозиторий на GitHub
───────────────────────────────────────
1. Зайдите на https://github.com
2. Нажмите "+" → "New repository"
3. Name: dota2-analyzer
4. Description: 🎮 Веб-приложение для анализа винрейта участников матчей Dota 2
5. Public или Private
6. НЕ добавляйте README, .gitignore, LICENSE
7. "Create repository"

ШАГ 2: Выполните команды в терминале
───────────────────────────────────────
Откройте PowerShell в папке C:\Analyzer

# Создайте коммит
git commit -m "Initial commit: Dota 2 Analyzer v1.0"

# Подключите GitHub (ЗАМЕНИТЕ YOUR-USERNAME!)
git remote add origin https://github.com/YOUR-USERNAME/dota2-analyzer.git

# Загрузите код
git branch -M main
git push -u origin main

# Создайте релиз
git tag -a v1.0.0 -m "Release v1.0.0"
git push origin v1.0.0

ШАГ 3: Настройте репозиторий на GitHub
───────────────────────────────────────
1. Settings → General → Добавьте описание
2. Добавьте Topics: dota2, golang, react, typescript
3. Actions → Разрешите workflows
4. Releases → Create release для v1.0.0
