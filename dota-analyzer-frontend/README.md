# Dota 2 Analyzer - Frontend

Современный веб-интерфейс для анализа винрейта участников матчей Dota 2.

## Технологии

- **React 19** + **TypeScript**
- **Vite** — быстрая сборка
- **Tailwind CSS** — утилитарные стили
- **Server-Sent Events (SSE)** — стриминг результатов в реальном времени

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение откроется на `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Особенности

### 🎨 Дизайн
- Темная тема в стиле Dota 2
- Адаптивный интерфейс (мобильные + десктоп)
- Цветовая индикация винрейта:
  - 🟢 Зеленый — WR ≥ 60%
  - 🟡 Желтый — WR ≥ 50%
  - 🔴 Красный — WR < 50%
  - 🟣 Фиолетовый — закрытый профиль

### ⚡ Производительность
- Результаты загружаются по мере обработки (SSE)
- Прогресс-бар показывает текущий статус
- Скелетон-лоадеры для плавной загрузки

### 🔧 UX
- Интуитивная форма входа с подсказками
- Настройка количества матчей
- Детальная информация по каждому матчу
- Легенда для быстрого понимания цветов
